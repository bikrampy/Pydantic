# Python Pydantic Practice Materials

## Topics Covered

1. **Foundation** - Basic Pydantic model setup and usage
2. **Optional** - Handling optional fields and default values
3. **Type** - Type annotations and validation
4. **Fields** - Advanced field configuration and constraints
5. **Field Validator** - Custom validation for individual fields
6. **Model Validator** - Validation across multiple fields
7. **Validation Order** - Understanding Pydantic's validation sequence
8. **Model Composition** - Combining models and nested structures

## Getting Started

Each file in the `src/` directory demonstrates a specific Pydantic concept with practical examples.

### Prerequisites

```bash
pip install pydantic
```

### Usage

Run any example file:

```bash
python src/01-foundation.py
```

## Learning Path

Start with `01-foundation.py` and progress sequentially to build a solid understanding of Pydantic's validation and data modeling capabilities.
